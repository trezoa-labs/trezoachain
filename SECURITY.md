# Security Policy

## Reporting a Vulnerability

TrezoaChain takes security seriously. If you discover a vulnerability, please **do not** open a public GitHub issue.

**Contact:** security@trzledger.org

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your suggested fix (optional)

We will respond within 48 hours and work with you on a responsible disclosure timeline.
