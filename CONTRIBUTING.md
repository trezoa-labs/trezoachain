# Contributing to TrezoaChain

Thank you for your interest in contributing to TrezoaChain! 🎉

## How to Contribute

### 🐛 Report a Bug
Open a [GitHub Issue](../../issues/new?template=bug_report.md) with:
- A clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Environment details

### 💡 Suggest a Feature
Open a [GitHub Issue](../../issues/new?template=feature_request.md) with:
- The problem you're trying to solve
- Your proposed solution
- Why this would benefit TrezoaChain

### 📝 Improve Documentation
- Fork this repo
- Make your changes
- Submit a Pull Request

### 🌍 Translations
We welcome translations of documentation into any language. Create a folder `/docs/[language-code]/` and submit a PR.

## Community Guidelines

- Be respectful and constructive
- English is the primary language for technical discussions
- No spam, FUD, or price discussion in technical channels
- Security vulnerabilities → email developers@trzledger.org directly (do NOT post publicly)
