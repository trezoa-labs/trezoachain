## 🔐 What is TrezoaChain?

TrezoaChain is the first **Layer-1 blockchain built from the ground up with post-quantum cryptography**. While other chains bolt on security patches after the fact, TrezoaChain was architected from day one to be quantum-resistant — using **NIST-approved CRYSTALS-Kyber and CRYSTALS-Dilithium** at the protocol level.

Quantum computers aren't a future threat. They're an engineering timeline. TrezoaChain is ready.

## 🌐 Why Post-Quantum Now?

> *"Harvest Now, Decrypt Later"* — adversaries are already collecting encrypted blockchain data today, planning to decrypt it when quantum computers become powerful enough.

Every blockchain using ECDSA (Bitcoin, Ethereum, Solana, etc.) is vulnerable. TrezoaChain eliminates this risk at the protocol level — not as an add-on.

**NIST standardized post-quantum algorithms in 2024:**
- **CRYSTALS-Kyber** → FIPS 203 (Key Encapsulation)
- **CRYSTALS-Dilithium** → FIPS 204 (Digital Signatures)

TrezoaChain uses both, natively.
